字符串中最多的字符

发表于 2020-05-26 | 评论数：

给定一段字符串, 查找查找其中做的字符和个数.

方法一

思路

遍历所有的字符串, 记录每个字符出现的个数,然后取出最多的.

时间复杂度

o(n)

空间复杂度

o(n)

方法二 删除法

思路:

每次取出第一个元素, 然后删除全部相同的, 哪一次删除的最多, 就哪一个元素最多

时间复杂度

o(n)

空间复杂度

o(1)

js中的字符串是不可变的,所以替换的话, 会开辟一块新的内存, 之前的str无用了, 就要等gc

方法三:

思路

先排序, 然后找出最多连续的

时间复杂度

o(nlogn):来自与sort函数
取出最长的重复字符串: o(n)
